RESTful web services with Java (Jersey / JAX-RS).
This tutorial explains how to develop RESTful web services in Java.
It uses the JAX-RS reference implementation Jersey.
In this tutorial Eclipse 4.7 (Oxygen), Java 1.8, Tomcat 6.0 and JAX-RS 2.0 (with Jersey 2.11) is used.
