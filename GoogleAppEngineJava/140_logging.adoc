[[logging]]
== Logging

You can use https://www.vogella.com/tutorials/Logging/article.html[standard Java Logging] for logging on the App Engine. 
For example to define a logger in your class EmailServlet use the following.

[source,text]
----
include::res/logging/logdefine.txt[]
----

To log a message use the following. 

[source,text]
----
include::res/logging/uselogger.txt[]
----

You find your logs on the https://www.vogella.com/tutorials/GoogleAppEngineJava/article.html#adminconsole[Admin Console]. 
the For further information see https://www.vogella.com/tutorials/Logging/article.html[Java Logging Tutorial] and
http://code.google.com/appengine/docs/java/runtime.html#Logging[Logging on the GAE/J].
